 Learning Outcomes
    • Apply CUDA streams and concurrency to build a pre-fetch pipeline that overlaps data
       transfers with computation and quantitatively evaluate its performance benefits.
    • Design and implement GPU kernels for data-parallel workloads, including batched matrix
       multiplication and pixel-wise image processing, while validating correctness against CPU
       implementations.
    • Analyze and optimize GPU memory access patterns, including transforming data layouts
       (AoS → SoA) and using shared memory tiling to reduce global memory traffic.
